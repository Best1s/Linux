#文件读写 open   默认为 'r'
'r’只读模式，必须打开一个已有的文件，且只能执行读操作
‘r+’读+追加模式，可读可写,即随时都可进行读与写。（写为追加在文件末尾）
w’只写模式，打开即默认创建一个新的空文件,若打开的是已有文件，则清空文件，且只能执行写操作
w+’写读模式，打开创建新文件，因此需要先把内容写进去在读
a’追加模式,不能读即追加写。
a+’追加读写模式，则不能立即读，因为此时光标已经是文件末尾

close -- 关闭文件。跟你编辑器的 文件->保存.. 一个意思。
read -- 读取文件内容。你可以把结果赋给一个变量。
readline -- 读取文本文件中的一行。
truncate -- 清空文件，请谨慎使用该命令。
write('stuff') -- 将stuff写入文件。
seek() 方法用于移动文件读取指针到指定位置。

exit(0)  中止程序

如果这个 if else  永远都不被执行到,可在else语句后面使用一个 die  的函数，让它打印出错误信息
